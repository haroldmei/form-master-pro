Changelog

Track the evolution of FormMasterPro with our detailed version history

This changelog documents all notable changes to FormMasterPro. We follow Semantic Versioning for our release cycle.

v1.0.0 Expected Q1 2023

Initial public release with core functionality.

Features:

  • AI-powered form field detection
  • Data extraction from DOCX and JSON files
  • Intelligent field mapping
  • Form auto-fill capabilities
  • Save and reuse field mappings
  • Basic profile management
v0.9.1-beta December 15, 2022

Final beta release with stability improvements and bug fixes.

Improvements:

  • Enhanced form field detection accuracy
  • Improved performance for larger documents
  • Better error handling and user feedback

Bug Fixes:

  • Fixed issue with certain input types not being properly filled
  • Resolved memory leak during document processing
  • Fixed UI rendering issues in the extension popup
v0.9.0-beta November 10, 2022

Major beta release with expanded functionality.

Features:

  • Added support for DOCX file format
  • Implemented AI-powered field mapping
  • Added ability to save and reuse field mappings

Improvements:

  • Redesigned UI for better usability
  • Enhanced form analysis algorithm
v0.5.0-alpha September 5, 2022

Initial alpha release with basic functionality.

Features:

  • Basic form field detection
  • Support for JSON data files
  • Manual field mapping
  • Simple form filling
