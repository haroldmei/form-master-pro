Documentation

Learn how to install, configure, and use FormMasterPro with our comprehensive documentation.

Installation

Browser Extension

  1. Download the latest release from GitHub Releases
  2. Extract the ZIP file to a folder on your computer
  3. In Chrome, navigate to chrome://extensions
  4. Enable "Developer mode" in the top-right corner
  5. Click "Load unpacked" and select the extracted browser-extension folder
  6. The FormMasterPro icon will appear in your browser's extension toolbar

System Requirements

  • Operating System: Windows 10+ (preferred), macOS, or Linux
  • Node.js: v16.0.0 or higher (for development only)
  • Browser: Chrome, Edge, or Brave (Chromium-based)

Getting Started

Basic Usage

  1. Navigate to a webpage with a form you want to fill out
  2. Click the FormMasterPro icon in your browser toolbar
  3. Click "Analyze Current Form" to detect form fields
  4. Click "Load Data File" to import your data from a file
  5. Click "Auto Fill Form" to automatically populate the form fields

Supported File Formats

  • JSON: Structured data in JavaScript Object Notation
  • DOCX: Microsoft Word documents

Creating Data Files

FormMasterPro can import data from different file formats. Here's how to prepare them:

JSON Format

Create a JSON file with the following structure:

{
  "personal": {
    "firstName": "John",
    "lastName": "Doe",
    "email": "john.doe@example.com",
    "phone": "555-123-4567"
  },
  "address": {
    "street": "123 Main St",
    "city": "Anytown",
    "state": "CA",
    "zip": "12345"
  }
}

DOCX Documents

FormMasterPro can extract data from structured DOCX documents, including:

  • Regular paragraphs of text
  • Tables with labeled rows and columns
  • Formatted sections with headings

The extension will attempt to intelligently parse the document structure and extract relevant information.

Field Mapping

Sometimes, you may need to manually map data fields to form fields:

  1. Click "Data Mappings" in the extension popup
  2. Use the mapping interface to connect your data fields with form fields
  3. Save your mappings for future use

FormMasterPro will remember your mappings for each website, making future form filling even faster.

For Developers

Building the Browser Extension

# Navigate to the browser-extension directory
cd browser-extension

# Install dependencies
npm install

# Development build with watch mode
npm run dev

# Production build
npm run build

# Package the extension
npm run package

Extension Architecture

The extension consists of several key components:

  • Form Analysis: Detects and categorizes form fields
  • Document Processing: Extracts data from various file types
  • Field Mapping: Connects data points to form fields
  • Form Filling: Populates form fields with the mapped data
