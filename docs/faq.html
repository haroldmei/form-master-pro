Frequently Asked Questions

Find answers to common questions about FormMasterPro

General Questions

What is FormMasterPro?

FormMasterPro is an AI-powered form automation tool that helps you fill out online forms automatically using data from various document formats. It analyzes form structures, extracts relevant data from your documents, and maps that data to the appropriate form fields.

Is FormMasterPro free to use?

Yes, FormMasterPro is an open-source project that's free to use. You can download it from our GitHub repository and use it without any cost. We may introduce premium features in the future, but the core functionality will remain free.

Does FormMasterPro work with all websites?

FormMasterPro is designed to work with most standard web forms. However, some websites with complex JavaScript forms, unusual form structures, or strong anti-automation measures may not be fully compatible. We're continuously improving compatibility with different websites.

Installation & Setup

How do I install FormMasterPro?

To install FormMasterPro:

  1. Download the latest release from our GitHub Releases page
  2. Extract the ZIP file to a folder on your computer
  3. Open Chrome and go to chrome://extensions
  4. Enable "Developer mode" in the top-right corner
  5. Click "Load unpacked" and select the extracted browser-extension folder

Why isn't FormMasterPro available on the Chrome Web Store?

We're currently working on getting FormMasterPro listed on the Chrome Web Store. The approval process takes time, especially for extensions with advanced capabilities. In the meantime, you can install it manually from our GitHub releases.

Does FormMasterPro work on browsers other than Chrome?

FormMasterPro is designed to work on Chromium-based browsers like Chrome, Edge, and Brave. Firefox and Safari are not currently supported, but we may add support for these browsers in the future.

Using FormMasterPro

What file formats does FormMasterPro support?

FormMasterPro currently supports:

  • JSON files
  • DOCX (Microsoft Word) documents

We're working on adding support for more formats including PDF, CSV, and plain text files.

How accurate is the form field detection?

FormMasterPro's field detection is highly accurate for standard form elements like text inputs, dropdowns, checkboxes, and radio buttons. The AI-powered field mapping improves over time as it learns from more examples. For complex or unusual forms, you may need to manually map some fields.

Can I save my field mappings for future use?

Yes, FormMasterPro automatically saves your field mappings for each website you visit. When you return to the same website in the future, your previous mappings will be loaded automatically, making the process faster each time.

Troubleshooting

FormMasterPro isn't detecting form fields correctly. What should I do?

If form detection isn't working properly:

  1. Make sure the page has fully loaded before analyzing the form
  2. Try refreshing the page and analyzing again
  3. Some dynamic forms may require clicking or interacting with the form first
  4. As a last resort, you can use the manual field mapping option

The extension is filling forms with incorrect data. How do I fix this?

If data is being mapped incorrectly:

  1. Click on "Data Mappings" in the extension popup
  2. Review and adjust the current field mappings
  3. Save your custom mappings for future use

How do I report a bug or request a feature?

You can report bugs or request features on our GitHub Issues page. Please check our bug reporting guidelines for instructions on providing the information we need to address your issue.
